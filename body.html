Internet:
It is a global data communication system
WWW(World Wide Web):
It is a repository of information (called resources) that can be accessed via Internet.
  • Tim Berners-Lee was the inventor of WWW











What is HTTP?

HTTP:Hyper Text Transfer Protocol.
  • HTTP is an application layer protocol for the WWW.
  • It is a 'generic stateless object-oriented protocol'.
  • It is a request-response protocol.




















HTML

  • HTML is a primary language used to encode documents cotaining hyperlinks.
  • Its first public version was released in 1991 by Tim Berners-Lee.
  • HTML got its final shape in May 2001,known as HTML 4.01
